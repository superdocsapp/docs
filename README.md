# SuperDocs Developer Documentation

Official developer documentation for [SuperDocs](https://superdocs.app) — the AI document editing platform.

## What is SuperDocs?

SuperDocs lets AI edit documents directly — changes happen inside the document, not in a chat window beside it. Give instructions in plain English, watch edits appear in real time.

**For individuals:** Use the [web editor](https://use.superdocs.app) to create and edit any type of document with AI assistance — contracts, reports, proposals, academic papers, and more.

**For developers:** Integrate AI document editing into your own product via REST API or MCP. One API, no prerequisites, production-ready today.
